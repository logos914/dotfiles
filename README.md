# Dotfiles de Nacho inspirado en fraruiz/dotfiles

## 🚀 Instalación

### ⚠️ Atención (para mac):

- Instalar [brew]("https://brew.sh/")


1. Ejeutar Instalador
   ```bash
   bash <(curl -s https://raw.githubusercontent.com/logos914/dotfiles/HEAD/installer)
   ```
2. Cambiar el shell a zsh
    ```bash
    chsh -s $(which zsh)
    ```

## 🔰 Actualizar
```bash
dot self update
```
